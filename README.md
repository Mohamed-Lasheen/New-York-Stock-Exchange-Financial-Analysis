# New York Stock Exchange Financial Analysis - Excel Dashboard

A comprehensive financial analysis of New York Stock Exchange (NYSE) data conducted entirely in Microsoft Excel. This project transforms raw stock market data into interactive dashboards and actionable insights using advanced Excel functionalities.

## üöÄ How to Use

1. **Download** the Excel file (`NYSE_Analysis_Dashboard.xlsx`)
2. **Enable Content** when opening to activate macros and interactive features
3. **Use Slicers** on the dashboard to filter by:
   - Time period (2010-2023)
   - Stock sectors
   - Individual companies
   - Performance thresholds
4. **Explore Different Tabs** for detailed analysis and raw data
